# Commit Message Convention
## Rules: 
**1. Follow this rule:**
```
<Icon>[<What Happend>]: <File Path>: <File Name>: <Description>
```
Example:
```
✨[Added]: website//package: model: `included`, `excluded` add to as model property
```

## Icon Rules: 
### [Added] => ✨
### [Updated] => ✅
### [Fix] => 🐛
